# Upload this patch to wuwa-database-server

1. Open the `wuwa-database-server` repository on GitHub.
2. Choose **Add file → Upload files**.
3. Open this extracted patch folder and select everything inside it, including `.github`, `wuwa_builder`, and `tests`.
4. Drag the selected items onto GitHub's upload page.
5. Commit directly to `main` with:

   `Fix official data discovery and reject empty releases`

This patch updates existing files and adds two test files.
